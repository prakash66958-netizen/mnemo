Why Mnemo

Four principles. Zero compromise.

shield_lock

Privacy first

All data lives on your device. No accounts, no telemetry, no servers. Even if we wanted your memories, we couldn't get them.

cloud_sync

Cloud sync

Real-time Firestore sync across devices. Works locally too — your data is always available even without internet.

bolt

Blazing fast

Isar-backed storage and debounced search that stays instant at 10k+ memories. Opens and responds in under a frame.

eco

Lightweight

Material 3 UI tuned for low-end Android. Sips battery, respects storage, refuses to be bloatware.

Features

Every thought has a home.

inbox

Universal save inbox

Text, links, notes, and shared content — a single infinite timeline. Pin what matters. Archive what's done. Swipe what's not.

share

Android share target

Share from WhatsApp, Chrome, Twitter, Telegram — any app. One tap, saved, classified.

location_on

Exact location pins

Attach GPS coordinates to any note. Share a precise pin — not a search URL — so others land on the exact spot.

handshake

Promise detection

"I'll send tomorrow at 5pm" and "Kal bhej dunga" both parse. Mnemo extracts the time and offers a reminder.

alarm

Smart reminders

Scheduled via local notifications. Fires even in airplane mode. Never misses a follow-up.

search

Instant search

Full-text search with category filters. Debounced, indexed, snappy. Find anything in a single breath.

task_alt

Daily habit tracker

Track habits with emoji, custom colors, and daily streaks. Water, reading, meditation — all in one tap.

dark_mode

Dark mode · export · clear

Full Material 3 dark theme. JSON export / import for portability. Nuclear "clear all" whenever you want a blank slate.

translate

Built for bilingual minds

The classifier ships with dictionaries for both English and Hinglish. "Remind me tomorrow" and "kal yaad dila dena" are the same thing to Mnemo.
